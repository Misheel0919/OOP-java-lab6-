# Print Interface

Print хйихэд хэрэглэгчид харагдах интерфейс.

# Интерфэйсийн бүтэц болон ажиллагаа
* Java
* Java Swing (JFrame, JPanel, JLabel, JButton, JComboBox)

* Зүүн талд - Preview хэсэг - энд '+ Enter file' гэсэн бичиг харагдах ба хэвлэх файл байрлах хэсэг.
* Баруун талд - Print тохиргоо - (JComboBox)
        -> Destination (Microsoft Print to PDF) 
        -> Pages (All, Current Page, Custom)
        -> Layout (Portrait, Landscape)
        -> Colour (Colour, Black & White) гэсэн сонголтууд байрлана
* Print товч дээр дарахад  консоль дээр ямар тохиргоогоор хэвлэгдэж байгааг дамжуулна.
        -> Printed! гэдэг мессэж хэвлэнэ.
* Cancel товч дээр дарахад  консоль дээр Cancelled гэж дамжуулна.
        -> Cancelled! гэдэг мессэж хэвлэнэ.
